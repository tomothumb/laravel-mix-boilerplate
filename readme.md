# Laravel Mix Boilerplate

This repository is a boilerplate of Laravel Mix for standalone ( without Laravel ) website.


## Dependency
"laravel-mix": "^2.1.11"

# Usage
```
$ git clone https://github.com/tomothumb/laravel-mix-boilerplate.git
$ cd laravel-mix-boilerplate

$ npm run prod
$ npm run dev
$ npm run watch
$ npm run hot

```
